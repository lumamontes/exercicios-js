## Como executar 

Clone esse projeto

```bash
$ git clone https://github.com/lumamontes/exercicios-js.git
```

Para iniciá-lo, siga os passos abaixo:
```bash
# Instalar as dependências
$ yarn
# Iniciar o projeto
$ yarn dev
```
